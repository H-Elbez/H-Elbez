Réseaux de neurones à impulsions

Les SNN sont moins efficaces que les réseaux de neurones classiques
  • Il nous manque une manière unifiée de créer et de former ce type de réseau
  • Absence d’une meilleure compréhension de l’activité lors de l’évolution du réseau

Conclusion

  • Les simulateurs fournissent des visualisations pour la surveillance, pas pour l'analyse

  • Nous avons mis l’accent sur la séparation du processus de simulation de celui de visualisation car les deux nécessitent des ressources considérables.
  • Les visualisations proposées par les simulateurs sont à des fins de surveillance et non d’analyse.
  • Il existe un besoin d’outils dédiés pour fournir une analyse visuelle de l’activité dans les SNN, des outils qui peuvent prendre en charge une analyse à grande échelle et fournir un degré d’interactivité et de support pour l’analyse dans le temps.

Contribution 3 :

Compression progressif pour SNN

Compression dans les réseaux de neurones à impulsions

  • Les synapses dans un réseau évoluent avec le nombre de neurones
  • La réduction des synapses génère un réseau plus petit


Types d’élagage
Élagage des synapses, Élagage des filtres, Élagage basé sur les neurones

  • Concentration sur le traitement des synapses élaguées, quand élaguer ou sur les critères d’élagage
  • Utilisation d'un seuil statique

L'utilisation d'un seuil dynamique
Les impulsions sont le moyen de communication dans les SNN. Ils sont transmis par les synapses, représentant le canal de communication entre les neurones. Le nombre de synapses dans un réseau évolue avec le nombre de neurones, donc avoir un moyen efficace de les utiliser conduit à un réseau plus économe en énergie et diminue le coût de production sur le matériel.
  • Élagage des poids : qui concerne la suppression des connexions synapses inutiles dans le réseau en fonction d’un seuil statique.
  • Élagage des filtres : cela supprime les filtres au lieu des synapses pour réduire le temps de calcul.
  • Élagage basé sur les neurones : nous exécutons l’opération d’élagage sur les neurones, ce qui a un impact négatif sur le réseau.
A partir de la littérature, nous pouvons voir que :
  • Les travaux existants se concentrent sur la manière de traiter les synapses élaguées, quand appliquer l’élagage ou bien sur les critères d’élagage.
  • Le même seuil statique est sélectionné sur la base d’expériences.

Compression dans les réseaux de neurones à impulsions



  • On peut retirer les synapses à un stade précoce
  • Apprentissage aprés élagage

Compression progressif pour SNN

Élagage progressive
$$ T_{n+1} = T_n + \alpha * (C_{rn} / C_n) \quad n \in \mathbb{N}$$
$\alpha$ : seuil initial
$C_{rn}$ : nombre de synapses restantes
$C_n$ : nombre total de synapses

Renforcement dynamique des poids synaptiques
$$ W_{n+1} = W_n + \beta * T_n, \quad n \in \mathbb{N}, \quad W \in [0,1]$$
$\beta$ : constante définie
$W_n$ : poids actuel
$T_n$ : seuil actuel

Objectif du renforcement synaptiques
  • Le maintien de l’activité moyenne des impulsions dans le réseau
  • Un phénomène qui existe aussi en biologie (synaptotrophins et synaptotoxins [1])
[1] Sanes JR, Lichtman JW, "Development of the vertebrate neuromuscular junction". Annual Review of Neuroscience 1999 22:1, 389-442

Sélection des paramètres


α = 0.05
β = 0.10
Les paramètres sélectionnés peuvent changer en fonction du type de réseau, du jeu de données utilisé et des objectifs choisis
